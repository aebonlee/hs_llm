### 2. 문제 해결 요구사항

**ErrorBoundary 충돌 문제:**
```
"문제가 발생했습니다 예상치 못한 오류가 발생했습니다. 불편을 드려 죄송합니다."
```
- localStorage의 기존 테마 데이터와 새로운 테마 시스템 충돌
- 마이그레이션 로직 필요

**다크모드 가독성 문제:**
```
"다크모드는 조금 과한 어두운 컬러이고 변경이 정확하게 안된 부분도 있네"
```
