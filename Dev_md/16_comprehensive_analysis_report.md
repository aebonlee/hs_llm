### 💡 핵심 기능 분석

#### 1. 강의계획서 생성기
- **입력**: 과목명, 학점, 수업시간, 학습목표
- **처리**: Bloom's Taxonomy 기반 학습목표 구조화
- **출력**: 15주차 완전한 강의계획서
- **특징**: CLO (Course Learning Outcomes) 자동 생성

#### 2. 루브릭 빌더
- **입력**: 과제명, 평가항목, 배점
- **처리**: 4단계 평가 기준 자동 생성
- **출력**: 체계적인 평가 루브릭
- **특징**: 가중치 기반 점수 계산

#### 3. 과제 생성기
- **입력**: 과목명, 주제, 난이도
- **처리**: 학습목표 연계 과제 설계
- **출력**: 상세한 과제 설명서
- **특징**: 난이도별 차별화

#### 4. 피드백 생성기
- **입력**: 학생명, 과제, 성과
- **처리**: 개인화된 피드백 생성
- **출력**: 건설적 피드백 메시지
- **특징**: 톤 선택 가능 (격려/건설적/상세)
